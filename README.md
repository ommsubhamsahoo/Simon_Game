# 🎮 Simon Game

A simple and fun **Simon Memory Game** built using **HTML, CSS, and JavaScript**.  
The game tests your memory skills by challenging you to remember and repeat an increasing sequence of colors.

---

## 🚀 Features

- Interactive color buttons
- Increasing difficulty with each level
- Smooth button flash animations
- Keyboard start functionality
- Score/level display

---

## 🛠️ Tech Stack

- **HTML** – Structure of the game  
- **CSS** – Styling and animations  
- **JavaScript** – Game logic and DOM manipulation  

---

## 🎯 How to Play

1. Press any key to start the game.
2. Watch the color sequence carefully.
3. Click the buttons in the same order.
4. Each level adds a new color to the sequence.
5. If you click the wrong color, the game ends.
